CSS Properties

Property Values Description
color red #FF00DD styles the font/text color
background-color name #FF000DD hsa rgb styles the background color
font-size px % styles the font/text size
opacity 0.25 50% styles how opaque, 0 = transparent, 1 = fully opaque
border-radius px % 50% = circle, can shape each corner
clear left right both none specifies how elements "bump" into each other
float left right move element as far left/right within the container
display block inline inline-block inline-block and block can assign width/height
position static relative absolute fied sticky change default position of an element
